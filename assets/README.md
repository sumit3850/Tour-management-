# assets

Put your company logo here as **`logo.png`** (or `logo.jpg` — then update the
reference in the quotation). The generated quotation looks for `assets/logo.png`
and shows it at the top of every quote.

If the file is missing, the quotation falls back to a styled text logo
("Island Explorer · BIRDING TOURS"), so quotes always look branded.

Recommended: a transparent-background PNG, roughly 600–1000px wide.
